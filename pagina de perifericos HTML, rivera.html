Tipos de Perifericos

Entre los perifericos mas conocidos estan:

  • Perifericos de salida
  • Perifericos de entrada
  • Perifericos de almacenamiento
  • Perifericos de comunicasion

Perifericos de salida

Proyectan la informacion desde el interior de un sistema informatico hacia el exterior. La informacion proyectada es, por lo general, en formato auditivo, visual o impreso.

EJEMPLOS:

  • Pantalla
  • Impresora
  • Auriculares
  • Altavoces

Perifericos de entrada

Son todos aquellos dispositivos que permiten introducir datos o informacion en una computadora para que esta los procese u ordene.

EJEMPLOS:

  • Teclado
  • Mouse
  • Escaner
  • Microfono

Perifericos de almacenamiento

Estos sirven para almecenar datos que el usuario quiera tener durante un largo de plazo, estos dispositivos guardan la informacion.

EJEMPLOS:

  • Discos Duros
  • Memorias USB
  • Disco DVD/CD
  • Disco duro portatil

Periofericos de comunicasion

Son todos aquellos que permiten la interaccion entre dos o mas ordenadores o entre un ordenador y algun otro dispositivo periferico.

EJEMPLOS:

  • Tarjera de red
  • Router
  • Switch
  • Bluetooth
